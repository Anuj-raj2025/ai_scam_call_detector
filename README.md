# 🎤 AI Scam Call Detector

## 🚀 Overview
This project detects scam calls in real-time using speech-to-text and intelligent keyword analysis.

## 🔥 Features
- Live voice detection
- Scam classification (Safe / Suspicious / Scam)
- Risk score (%)
- Reason-based explanation

## 🛠️ Tech Stack
- Python (Flask)
- JavaScript (Speech Recognition API)
- HTML / CSS

## 📊 Workflow
Voice Input → Speech to Text → Text Processing → Backend Analysis → Scam Detection → Risk Score → Result Display

## ⚠️ Limitations
- Depends on speech accuracy
- Rule-based detection
- No direct phone call integration

## 🎯 Use Case
- Detect OTP scams
- Bank fraud calls
- Phishing attempts
